

# 手动标注

Ragomics的可视化界面提供手动标注功能。用户可通过聚类、染色、通过基因表达高低等方式筛选细胞并加入对应标注集。

Step 1: 进入可视化面板，选中annotation tag (cell set右侧)  
{{< image src="images/visual-panel-manual-annotation-0.png" alt="Alt text for the image" width="500px">}}  
Step 2: 点击加号，选择手动标注（Manual Annotation），创建一组新的annotation  
{{< image src="images/visual-panel-manual-annotation-create-annotation.png" alt="Alt text for the image" width="500px">}}  

Step3: 在细胞散点图/Embedding图像内通过各类筛选条件工具选择细胞，加入annotation。下图创建cell-type-1细胞集合  
{{< image src="images/visual-panel-manual-annotation-create-cell-type.png" alt="Alt text for the image" width="500px">}}  
点击保存后，加入的细胞集（cell-type-1）会被添加至右侧annotation中  

已标注的细胞被**置灰**，避免标注过程中被重复选取。


重复Step3，添加多种细胞类型  
{{< image src="images/visual-panel-manual-annotation-create-cell-type-result-2.png" alt="Alt text for the image" width="500px">}}  

标注所有细胞后，点击上传按钮  
{{< image src="images/visual-panel-manual-annotation-upload.png" alt="Alt text for the image" width="200px">}}  




- 左侧obs内分类（category）加入annotation
- 右侧细胞集合（cell set）加入annotation
