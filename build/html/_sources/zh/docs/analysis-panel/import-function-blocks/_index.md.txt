
# 独立算法功能块（Function Block）

## 创建算法节点（Node）
左侧栏选中单独分析功能栏
选中并拖入右侧分析面板中
{{< image src="images/zh/ap-fb-leftSideBar.png" alt="Alt text for the image" >}}


## 节点按钮
独立算法功能模块可与上传的单细胞测序数据或其他节点相连。选中一侧的连接部分拖动连接至另一功能块或数据节点即可。  
连接节点至数据节点或已产生数据的前置节点后可以运行节点，具体功能请参考下图所示
