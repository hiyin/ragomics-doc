计算子集标志基因Top gene/gene marker/hvg¶

  1. 进入可视化面板，点击可视化面板上方的Top gene按钮(下图所示)
    {{< image src="images/visual-panel-top-genes-DE.png" alt="Alt text for the image">}}

  2. 点击按钮后，选择可视化面板左侧category或右侧cell sets内的cell set，点击run即可在选中子集内计算gene marker。背景workflow中会在当前节点内自动添加计算节点
    {{< image src="images/visual-panel-top-genes-sample.png" alt="Alt text for the image">}}

  3. 计算完成后，workflow内进入相应的节点
    {{< image src="images/visual-panel-top-genes-result-flow.png" alt="Alt text for the image" width="400px">}}

  4. 可视化面板右下方出现Top genes
    {{< image src="images/visual-panel-top-genes-result-vp.png" alt="Alt text for the image" width="400px">}}
