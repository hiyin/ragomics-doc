富集分析（Gene set enrichment analysis）及细胞通路分析（Pathway analysis）¶

操作流程¶

  1. 打开可视化面板

  2. 面板内follow下图a中UI，点击new

  3. 弹出图b UI，填入、及

    • job name（合理名称即可；推荐以“-”间隔，例如2024-human-fibroblast）

    • 基因集（gene set）

    • 物种（species）

    • 选择数据库，数据库选项自动跟随物种改变

  4. 图b内点击运行按钮
